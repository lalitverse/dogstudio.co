# Dogstudio Inspired Clone

A modern creative agency website inspired by the design and user experience of Dogstudio.co.

## Features

- Modern responsive design
- Smooth scrolling experience
- Interactive animations
- Three.js / React Three Fiber integration
- Custom UI components
- Mobile-friendly layout

## Tech Stack

- React
- Vite
- GSAP
- Three.js
- React Three Fiber
- CSS

## Purpose

This project was created for educational and portfolio purposes to practice modern web development, animations, and interactive user experiences.
